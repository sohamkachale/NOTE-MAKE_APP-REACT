## ğŸ“– Overview

NotesApp is a next-generation notes management and productivity platform built with React. Designed for creators, students, and professionals, NotesApp offers rich functionality for note-taking, organization, and easy access across devices. The app features secure authentication, dark/light theming, instant search and sorting, and a stunning responsive interface powered by Framer Motion.

---

## âœ¨ Features

- **User Authentication:** Secure registration and login (username/password), with session persistence.
- **Notes Management:** Instantly create, edit, delete, and copy notes; everything is saved in the browser and tied to your account session.
- **Real-Time Search & Sort:** Filter notes by keyword and sort by update date, creation date, or title.
- **Theme Toggle:** Toggle light/dark appearance; preferences are saved automatically.
- **Animated, Responsive UI:** Fluid page transitions and smooth element animations via Framer Motion, fully mobile-friendly.
- **Enhanced User Experience:** Validation, clear error/success feedback, password visibility toggles, and intuitive gestures for efficient workflow.
- **Customization Ready:** All styling uses scalable CSS variables for easy personalization and extension.
